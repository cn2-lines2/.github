## 先搞懂：CN2线路是个啥

用最简单的比喻来说：

普通163骨干网就像国道，车多、路窄、晚高峰必堵。**CN2线路就是专门修的高速公路**，车道宽、路况好、走的人少，延迟稳定得多。

CN2是中国电信的第二代承载网（ChinaNet Next Carrying Network），但它自己还分两种：

**CN2 GT（Global Transit）**：半程CN2，出了国就走CN2，但回程可能会混入普通163骨干网。晚高峰还是会卡，只是比纯163好一点点。

**CN2 GIA（Global Internet Access）**：全程CN2，从头到尾都走高速。路由节点全走59.43开头的地址，延迟稳定，丢包率极低，是CN2里的顶配。

> **关键识别方法**：看traceroute，全走`59.43.x.x`节点才是正宗GIA；如果夹杂大量`202.97.x.x`，多半是GT或者纯163线路。

很多主机商喜欢玩文字游戏，只说"CN2线路"但不说是GT还是GIA。买之前一定要问清楚，否则花了GIA的价钱买了GT的体验。
